Constellations: Orion
This project visualizes the Orion constellation using Matplotlib in both 2D and 3D.

Overview
The notebook plots the provided star coordinates with scatter() to show how Orion looks from Earth and how the stars are positioned in space.

Files
orion_project.ipynb: Main notebook for the project.

script.py: Python code for the visualization tasks.

README.md: Project description and usage notes.

What the project does
Imports matplotlib.pyplot for plotting.

Imports Axes3D from mpl_toolkits.mplot3d for 3D plotting.

Creates a 2D scatter plot of Orion using the x and y coordinates.

Creates a 3D scatter plot using the x, y, and z coordinates.

Updates the plot styling to resemble a night sky.

Running the project
Open the notebook in Jupyter or Codecademy and run the cells in order. The plots will appear as the notebook executes.

Notes
The coordinate lists are already provided in the assignment, so no external data download is required for the basic project.
